# Reverse Mode
A mod for Neon White that lets you play levels backwards. 

### Important Notes
- This mod uses NeonLite's built in anti-cheat, so make sure you have the latest version of [NeonLite](https://github.com/Faustas156/NeonLite) installed.
- **Quick Restart must be disabled** for the mod to function. 
You can disable it in `NeonLite > Optimization.`
